# Zed UI Bileşenleri

Bu başlık GPUI çekirdeğinin parçası değildir; `ui` üst katman Zed UI bileşenlerini anlatır. Güncel ve detaylı bilgiye [Zed UI Bileşenleri](../bilesenler/bilesenler.md) bölümünden erişilebilir.

GPUI tarafında yalnız çekirdek primitive'ler, render modeli, pencere bağlamı ve düşük seviye API yüzeyi anlatılır. `Button`, `IconButton`, `List`, `PopoverMenu`, `TabBar`, `Picker` gibi Zed uygulama bileşenleri için ilgili bileşen bölümü ve reçeteler takip edilmelidir.
